# Welcome to Mod Data Center SFS
made to be a data source for SCAN

## How It Works?
 (1) SCAN -> (2) Mods Repo -> (3) xxxx.json

1- SCAN downloads the xxxxx.json through repo.json
2- The Repo has the repo.json where it points to each xxxx.json
3- The xxx.json has the mod data including the download link
